#SEGUNDO SISTEMA EM DELPHI

Este foi o segundo projeto desenvolvido como parte do meu aprendizado inicial em Delphi, com o objetivo de conhecer os componentes básicos da VCL e aprender a desenhar as interfaces gráficas dentro da IDE, sem integração com banco de dados e deixando a programação de eventos e regras para as próximas aulas.

A proposta da aula era exclusivamente explorar os componentes padrões, posicioná-los corretamente na tela entendendo suas propriedaddes principais e criando uma interface completa sem a necessidade de funcionalidades ativas.
Durante a criação da interface, utilizei diversos componentes básicos da VCL, como: TLabel, TEdit, TCombox, TRadioButton, TCheckBox, TMemo e TButton.
